## üèóÔ∏è Design System Rails

### UI Components
- **Corners**: Use `rounded-md` or `rounded-lg` for a modern, friendly feel.
- **Shadows**: Use `shadow-lg` for cards and floating elements to create depth.
- **Interactions**: Implement smooth hover transitions (`transition-all`) with slight scaling (`scale-105`) for interactive elements.
- **Layout**: Utilize **Glassmorphism** and **Bento Grid** layouts for dashboard views.

### Typography
- **Headings**: Use a bold, premium Serif font (like Inter or a Google Font equivalent).
- **Body**: Clean, high-readability Sans-Serif.
